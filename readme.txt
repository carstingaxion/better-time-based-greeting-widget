A Widget to show text messages depending on the current time.

== Description ==

With this widget you can show unlimited texts during the corresponding times to your visitors.

Inspired by the [time-based-greeting Plugin](http://wordpress.org/extend/plugins/time-based-greeting/), which gives you only the possibility to define exact four timeframes.
With this improved Version you're able to add unlimited timeframes.


== Installation ==

1. Extract the zip file
2. Drop the content in the wp-content/plugins/ or wp-content/mu-plugins directory of your WordPress installation
3. Activate the Plugin from Plugins page
4. Activate the widget by dropping it to an active sidebar under Appearance > Widgets
